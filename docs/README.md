# 기능 목록


## version : 1.2.0


### Application
- [x] 게임 시작 문자 을 출력한다. 


### InputView
- [x] 다리의 길이를 입력받는다.
- [x] 사용자가 이동할 칸을 입력받는다.
- [x] 사용자가 게임을 다시 시도할지 종료할지 여부를 입력받는다.
- [x] 사용자의 입력 유효성 검사를 하여 예외 처리 한다.



### BridgeMaker
- [ ] 위 칸과 아래 칸 중 건널 수 있는 칸은 0과 1중 무작위 값을 이용하여, 전체 다리를 생성하는 기능 
  - 무작위 값이 0인 경우 아래 칸, 1인 경우 위 칸이 건널 수 있는 칸이 된다.
